## Programs I use

#### Basic
*Operating system:*
I use [arch](https://archlinux.org/), btw.

*Terminal:*
I use [alactritty](https://alacritty.org/) as my main terminal and have [xterm](https://xtermjs.org/) as a backup.

*Shell:*
For casual use I use [fish](https://fishshell.com/) and for any shell scripting I use bash.

*Window manager:*
[xmonad](https://xmonad.org/)

*Text editor/IDE:*
I mostly use a distribution of emacs called [doom-emacs](https://github.com/hlissner/doom-emacs). For bigger projects I'll use [vscode](https://code.visualstudio.com/).
If I'm just quickly editing someting in the terminal like a config file I'll just use vi/vim, no need to pull up emacs for that.

#### Utilities
*File manager:*
I use a file manger less and less nowadays and just do everything from the terminal. If I do need a GUI file manger for some reason I use PCManFM.

*Web browser:*
[Firefox](https://www.mozilla.org/en-US/firefox/new/), obviously.

*Mail client:*
I stated to use the built in email client in emacs more but otherwise I use [thunderbird](https://www.thunderbird.net/en-US/).

*Video player:*
Used to use vlc, now I use mpv like all the cool kids.

*Torrent client:*
[qBittorrent](https://www.qbittorrent.org/).

#### Production
*Video and audio:*
I use [OBS Studio](https://obsproject.com/) to record video and [Kdenlive](https://kdenlive.org/en/) to edit videos.

*Digital Art:*
Been getting into digital art lately and have been loving [krita](https://krita.org/en/).

*Writing documents:*
Basic documents I write in markdown and more advanced stuff I write in LaTeX and compile to pdf. Emacs is also great for editing LaTeX docs, especially with live-latex-preview.

*Office suite:*
[LibreOffice](https://www.libreoffice.org/). I pretty much only use an office suite for the spreadsheet program and sometimes the presentation program but I use org-mode in emacs for that now, so looking for a better way to do spreadsheets so I don't need an entire office suite just to make spreadsheets.

*PDF viewer:*
[zathura](https://pwmt.org/projects/zathura/)

*Image modification:*
[GIMP](https://www.gimp.org/)
